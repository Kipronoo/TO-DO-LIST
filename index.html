TO-DO LIST

The TO-DO LIST application is a productivity tool designed to help users manage their tasks efficiently. The project features a user-friendly interface that allows users to add, edit, and organize their to-do items. Key functionalities include task filtering, push notifications, and integration with third-party APIs like SendGrid for email notifications and Twilio for SMS alerts.

Key Features

Task Management

Effortlessly create, edit, and delete tasks with a user-friendly interface. Organize tasks by setting due dates, priorities, and categories. Keep track of your progress with a clear overview of completed and pending tasks.

Task Filtering and Sorting

Easily filter and sort tasks by various criteria, such as priority, due date, and category. Use powerful search capabilities to quickly find specific tasks, ensuring you stay focused and organized.

About This Project

This project was inspired by my personal struggle to manage tasks efficiently across different areas of life. As a software engineering student, balancing school assignments, personal projects, and daily tasks became overwhelming. Traditional task management tools often lacked flexibility or features that suited my needs, so I decided to build a tool that would provide a more intuitive way to organize tasks. The development journey for the TO-DO LIST application began with identifying the core features I needed to stay productive: clear task management, efficient filtering, and timely notifications. Over several weeks, I worked on implementing these features while also improving my skills in web development, API integration, and user interface design. This project is a portfolio piece, developed as part of my learning journey at Holberton School, where I explored both front-end and back-end development technologies.
