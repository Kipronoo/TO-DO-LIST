TO-DO LIST

The TO-DO LIST application is a productivity tool designed to help users manage their tasks efficiently. The project features a user-friendly interface that allows users to add, edit, and organize their to-do items. Key functionalities include task filtering, push notifications, and integration with third-party APIs like SendGrid for email notifications and Twilio for SMS alerts. The application also supports Google OAuth for secure user authentication. The project showcases a clean and intuitive design, with features aimed at enhancing task management and user experience.
